# Portfolio Moderne - Fanirianjatovo

Un portfolio web moderne et responsive présentant mes compétences en développement web full-stack.

## 🚀 Aperçu

Ce portfolio présente mes projets, compétences et expériences en tant que développeur web. Il est conçu avec une approche moderne utilisant les dernières technologies web pour une expérience utilisateur optimale.

## 📋 Fonctionnalités

- **Design Responsive** : Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Mode Sombre/Clair** : Basculement entre les thèmes avec sauvegarde des préférences
- **Navigation Fluide** : Scroll smooth et animations au scroll
- **Formulaire de Contact** : Fonctionnel avec validation
- **Projets Interactifs** : Liens directs vers les dépôts GitHub
- **Animations Modernes** : Transitions et effets hover élégants

## 🛠️ Technologies Utilisées

## 🚀 Installation et Utilisation

### Prérequis

- Un navigateur web moderne
- Connexion internet (pour les polices Google Fonts)

### Installation Locale

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/Fanirianjatovo/Myportfolio.git
   cd Myportfolio
   ```

2. Ouvrez `index.html` dans votre navigateur ou utilisez un serveur local :

   ```bash
   # Avec Python
   python3 -m http.server 8000

   # Avec Node.js
   npx serve .

   # Avec PHP
   php -S localhost:8000
   ```

3. Accédez à `http://localhost:8000`

## 🎨 Personnalisation

Les couleurs sont définies via des variables CSS dans `:root` :

